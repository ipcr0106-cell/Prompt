서술형

                        

                            재사용: 낮음
                            유연성: 높음
                        

                    

                    
맥락과 배경을 설명하여 AI가 상황을 이해하도록 유도

지침형

                        

                            재사용: 중간
                            유연성: 중간
                        

                    

                    
단계별 절차나 명확한 규칙을 제시하여 원하는 출력을 유도

                    

                        

                            예시 프롬프트

                            재사용: 높음
                            유연성: 낮음

4. 상호 합의: 지연된 물량에 대한 차기 주문 시 할인 또는 보험료 지원

Narrowing: 지연 배상금(L/D) 청구 대신,

APE 마스터 프롬프트

                    
역할: 당신은 프롬프트 엔지니어링 전문가입니다
주어진 작업에 대해 최적의 프롬프트를 자동으로 생성·평가·개선합니다

APE 실행 순서

STEP1: 후보 프롬프트 생성
아래 작업을 위한 프롬프트 후보 10개를 생성합니다
각 후보는 서로 다른 전략을 사용해야 합니다

[전략 유형]
- 역할 부여형: "당신은 xxx 전문가입니다"
- 단계별 지시형: "step1: … step2: …"
- 예시 포함형: 퓨샷 방식으로 패턴 제시
- 제약 명시형: "반드시 ~하되, ~하지 마세요"
- 목적 중심형: "최종목표는 ~이며 …"
- 독자 설정형: "~을 모르는 사람도 이해하도록"
- 형식 지정형: "결과를 [형식]으로 출력하세요"
- 품질 기준형: "~한 기준으로 평가했을 때 90점 이상…"
- 역방향형: "나쁜 예시는 ~이고, 좋은 예시는 ~"
- 메타 지시형: "이 작업의 본질은 ~이며…"

STEP2: 후보 자체 평가 (각 100점 만점)
- 명확성(25점): 지시가 모호하지 않은가
- 완전성(25점): 필요한 정보가 다 있는가
- 실행가능성(25점): AI가 바로 실행하는가
- 품질유도력(25점): 높은 품질의 결과를 이끌어 내는가

STEP3: 상위 3개 선택 후 교차 최적화
상위 3개 프롬프트의 장점을 결합하여 최종 최적 프롬프트 1개 생성

STEP4: 최적 프롬프트 출력
- 최적 프롬프트 전문
- 왜 이것이 최선인지 설명
- 사용 시 주의사항

작업입력:
작업: {고객 클레임 이메일 답장 작성}
원하는 결과물: {고객이 화가 풀리고 우리 회사를 신뢰하게 하는 답장}
피해야 할 것: {방어적 태도, 책임 회피, 공허한 사과}
